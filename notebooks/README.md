# Training Notebooks

Kaggle notebooks for model training (Week 1-2).

## Notebooks List

1. `00_setup_environment.ipynb` - Environment setup
2. `01_prepare_datasets.ipynb` - Dataset preparation
3. `02_train_classifier.ipynb` - Classification model
4. `03_train_extractor.ipynb` - Extraction model
5. `04_train_ner.ipynb` - NER model
6. `05_train_summarizer.ipynb` - Summarization model
7. `06_benchmark_models.ipynb` - Model benchmarking

## Training Platform

All training done on Kaggle (P100 GPU, free tier).
